### Formal definition

It is not hard to use category theory to define the data platform in the abstract level, but it is hard to classify the associated functor on the system.

From abstract point of view, our system is a transition system. There are some research works about complex system have been done, which naturally cover the transition system, for example, [Complex software systems : Formalization and Applications](https://centralesupelec.hal.science/hal-00782879/file/aiguier_al-complex-software.pdf) and [Towards a Definition of Complex Software System](https://arxiv.org/pdf/2306.11817).

We will continue to study and research in this area.
