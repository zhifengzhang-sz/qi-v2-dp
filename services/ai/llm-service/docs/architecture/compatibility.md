- Python 3.10+
- Docker 20.10+

- CPU: x86_64 Linux

- HuggingFace Hub models
- PyTorch models
- Supported formats: SafeTensors, BIN

- RAM: 16GB minimum
- Storage: 20GB minimum

### GPU Mode

- GPU RAM: 24GB recommended
- System RAM: 32GB minimum
- Storage: 50GB minimum
